Use searchable names

  • Create const variables
  • Consts also save from regression bugs by providing more context information
  • Too short names are hard to search

Avoid encodings

  • Hungarian notation
    • Useful for languages that don't have good type system
    • When we change type, we need to change variable name too
  • Member variables prefixes and suffixes
  • Interface names beginning with "I"

Mental mappings

  • Usage of i,j,k for loop indices
  • Class names should be nouns
  • Function names should be verbs
    • getXXX()
    • setXXX()
    • isXXX()
  • Use factory constructors instead of overloading 
                              
Complex fulcrumPoint = Complex.fromRealNumber(23.0);
Complex fulcrumPoint = new Complex(23.0);

private void printGuessStatistics(char candidate, int count) {
    String number;
    String verb;
    String pluralModifier;
    if (count == 0) {
        number = "no";
        verb = "are";
        pluralModifier = "s";
    } else if (count == 1) {
        number = "1";
        verb = "is";
        pluralModifier = "";
    } else {
        number = Integer.toString(count);
        verb = "are";
        pluralModifier = "s";
    }
    String guessMessage = String.format(
        "There %s %s %s%s", verb, number, candidate, pluralModifier
    );
    print(guessMessage);
}
